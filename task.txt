تفاوت‌های اصلی بین کامپیوترهای تو برد Arduino و Raspberry Pi در موارد زیر  است:


Arduino از یک میکروکنترلر AVR یا ARM به عنوان قلب خود استفاده می‌کنند. این میکروکنترلرها معمولاً دارای حافظه و پردازشگر محدودتری نسبت به یک کامپیوتر کامل هستند. Arduino‌ها بیشتر برای پروژه‌هایی که نیاز به کنترل ساده سخت‌افزار دارند، مورد استفاده قرار می‌گیرند


Raspberry Piها از یک پردازنده ARM قدرتمند به همراه حافظه بزرگ‌تر استفاده می‌کنند. این‌ها در واقع یک کامپیوتر کامل هستند که از سیستم‌عامل‌های متنوعی مانند Linux پشتیبانی می‌کنند. Raspberry Pi‌ها قادرند برنامه‌های پیچیده‌تر و همچنین کارهای مرتبط با شبکه و اینترنت را اجرا کنند.

برای Arduino‌ها معمولاً از زبان برنامه‌نویسی C/C++ و محیط توسعه Arduino IDE استفاده می‌شود. 


Raspberry Piها دارای پورت‌های متنوعی برای اتصال به دستگاه‌های خارجی مانند HDMI (برای نمایشگرها)، USB (برای اتصال دستگاه‌های جانبی)، Ethernet (برای شبکه) و GPIO (برای اتصال به سخت‌افزار خارجی مثل Arduino) هستند. این قابلیت‌ها باعث می‌شود Raspberry Pi‌ها مناسب برای کاربردهای گوناگون از جمله سرور خانگی، اتصالات اینترنت اشیاء (IoT)، رباتیک و غیره باشند.

Raspberry Pi‌ها قابلیت اجرای برنامه‌های بسیار گسترده‌تر را دارند و می‌توانند با استفاده از زبان‌های برنامه‌نویسی گوناگونی مانند Python، Java، C/C++ و زبان‌های دیگر برنامه‌نویسی شوند

Arduino‌ها بیشتر برای پروژه‌های الکترونیکی و کنترل سخت‌افزاری ساده مورد استفاده قرار می‌گیرند، مانند سیستم‌های خودکار ساده، سنسورها، روباتیک کوچک و غیره.

Raspberry Pi‌ها برای کاربردهای پیچیده‌تری مورد استفاده قرار می‌گیرند از جمله سرور خانگی، سیستم‌های IoT، مدیریت سیستم‌ها، پخش ویدیو، ایجاد محیط‌های آموزشی، ایجاد وب‌سایت محلی، و غیره.
